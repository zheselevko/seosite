1. Стоимость продвижения ресторана
Цели поискового продвижения

Сайт ресторана должен работать на имидж заведения и продавать через Интернет. Именно эту цель мы выбираем для продвижения Вашего ресторана. Для этого необходимо:

  • Подготовить каталог для сайта ресторана - структуру движения потенциального клиента
  • Регулярно наполнять сайт продающими страницами
  • Начать закупку вечной ссылочной массы Провести информационную кампанию в социальных сетях
Коммерческое предложение на продвижение ресторана

Для достижения поставленных целей продвижения ресторана наша команда аналитиков подготовит для Вас аппетитное предложение по привлечению клиентов из поиска. В нем мы обязательно отразим:

  • Показатели по трафику из поисковых систем, которые и будет занесены в Договор
  • Инвестиционные потребности для создания продающих страниц на сайте Вашего ресторана
  • Показатели по уровню вложений в вечную ссылочную массу
Договор на продвижение ресторана

После проведения согласования коммерческого предложения мы подписываем с Вами Договор на продвижение Вашего ресторана, в котором гарантируем ключевые показатели для сайта Вашего ресторана. В договоре указаны:

  • Показатели сайта Вашего ресторана по количеству переходов
  • Цена продвижения ресторана
  • Количество новых имиджевых страниц, которые будут созданы
2. Запуск продвижения ресторана
Сбор полного ядра запросов для сайта Вашего ресторана

Для успешного продвижения сайта ресторана мы собираем наиболее полное ядро запросов, по которым Вас могут искать потенциальные посетители. Для сайта Вашего ресторана учтем низкочастотные и высокочастотные запросы.

  • Начать закупку вечной ссылочной массы Провести информационную кампанию в социальных сетях
  • Подбор хедлайнеров (например, пицца, суши, «Цезарь» и т.д.)
  • Формирование наиболее полного ядра запросов по хедлайнерам (технология Зекслер по сбору запросов из вордстат, подсказок, выдаче поиска, релевантных подсказок)
  • Группируем запросы по продающим свойствам Алгоритмическая отчистка (технология Зекслер) ядра от «мусорных» запросов
  • Ручная фильтрация запросов нашим аналитиком
ТЗ для развития внутренней инфраструктуры сайта ресторана

Полное ядро запросов приводим к структуре (каталогу) с удобной навигацией пользователей по ней, а также создаем детальное задание для копирайтеров по работе с инфраструктурой сайта (наполнению его продающим контентом). Составим ядро с учетом особенности Вашего заведения и кухни, которую Вы предлагаете.

  • Разработка структуры каталога для сайта
  • Формирование ТЗ для контент менеджеров для проведения работ с категориями каталога
  • Подготовка ТЗ для копирайтеров для продающих статей
  • Написание ТЗ для внутренней перелинковки страниц сайта
Создание ТЗ по работе над имиджем Вашего ресторана на сторонних ресурсах

На основе разработанного ядра запросов и планируемой структуры сайта ведем работу направленную на увеличение положительных упоминаний ресторна в Интернете. Для этого мы делаем:

  • Создаем список естественных анкоров для каждой страницы сайта, с целевым действием
  • Разрабатываем требования к таргетингу (по тематике, по регионам, показателям) сторонних ресурсов для размещения имиджевых статей
  • Формирование ТЗ для закупки вечной ссылочной массы
  • Проработка информационной кампании в социальных медиа
Подготовка к наполнению сайта

Работа с каталогом и меню для его вывода (проверка, оптимизация, доработка при необходимости)

Проверка перечня типов материалов (например, полезные статьи/блог/новости) для размещения продающих страниц под запросы потенциальных клиентов и меню для его вывода

Получение от технических специалистов сайта характеристик особенностей работы с каталогом и продающими страницами

При необходимости – разработка, дизайн и внедрение данных решений командой внешней разработки Зекслер

3. Продвижение ресторана
Работа с внешними источниками

Для сайта важно, чтобы поисковики знали о его существовании и он был аккуратно подтянут в поиск вместе с его содержимым. В связи с этим необходимо увеличить присутствие сайта в Интернете наиболее вечными методами:

  • Закупаем вечные ссылки на соответствующие продающие страницы
  • Размещаем имиджевые статьи на тематических ресурсах (плюс целевой трафик)
  • Ведем агрессивную поддержку в социальных сетях: в комментариях, на страницах пользователей и сообществ
Развитие внутренней инфраструктуры сайта ресторана

Развиваем и наполняем сайт Вашего ресторана под привлечение клиентов из поиска, делаем Ваш сайт максимально удовлетворяющим запросам Ваших потенциальных клиентов, а значит и поисковика. Раскрываем все ключевые преимущества Вашего ресторана и акцентируем внимание на них.

  • Создаем новые продающие страницы в каталоге сайта
  • Размещаем имиджевые и продающие статьи под запросы потенциальных клиентов - полезные статьи
  • Рерайтинг существующих посадочных страниц
  • Внутренняя перелинковка посадочных страниц на сайте
Контроль эффективности

Мы еженедельно подводим итоги продвижения ресторана, изучаем показатели сайта и привлечения клиентов из поиска. Эта аналитика содержит:

  • Изменения позиций по ядру запросов для Яндекс, Google и Mail
  • Движение и рост трафика из поиска (план/факт анализ)
  • Ссылки на статьи с которыми проводилась работа:вновь созданные и оптимизированные продающие страницы для сайта Вашего ресторана
